# Vivian Wang | 王维 - 个人品牌网站

> 产业数字化战略专家 & 资本运营合伙人的专业形象展示平台

## 🎯 项目概述

这是一个专为高端专业人士打造的双语个人品牌网站，全方位展示**Vivian Wang（王维）**作为产业数字化战略专家与资本运营合伙人的独特价值主张、核心能力与职业成就。网站设计参考了国际顶尖建筑事务所KPF的设计美学，采用极简主义风格，提供优雅的视觉体验和流畅的交互体验。

## ✨ 核心特色

### 1. **KPF风格的国际化设计**
- **极简主义美学**：黑白灰为主色调，金色点缀，传递专业与高端品质
- **大图展示风格**：高质量专业照片全屏展示，建立强烈的视觉冲击

- **现代字体系统**：Inter（英文）+ Noto Sans SC（中文）完美结合

### 2. **中英文双语无缝切换**
- **一键切换**：导航栏语言按钮实现中英文即时切换
- **智能记忆**：自动保存用户的语言偏好到本地存储
- **全站覆盖**：所有文本内容均支持双语切换
- **体验流畅**：切换过程平滑自然，无页面刷新

## 🎨 设计理念

### KPF设计风格参考
本网站参考了国际顶尖建筑事务所KPF（Kohn Pedersen Fox）的网站设计理念：

1. **极简主义**
   - 黑白灰为主色调
   - 金色作为唯一强调色
   - 大量留白营造呼吸感

2. **大图展示**
   - 高质量图片全屏展示
   - 灰度滤镜保持专业感
   - 悬停交互增加活力

3. **清晰层次**
   - 细线分隔不同区块
   - 编号系统增强识别
   - 卡片布局统一规范

4. **专业排版**
   - 现代无衬线字体
   - 精确的行高与间距
   - 多语言完美呈现

### 如何使用
